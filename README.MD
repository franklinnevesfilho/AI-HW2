# Usage
```
Usage: python main.py <problem_name> <attribute>
    Problems:
        - island_count:
            attributes: dfs, bfs
        - sliding_puzzle:
            attribute: default, or grid format: [1-2-3][4-5-6][7-8-0]
    
    Example for sliding_puzzle:
        python main.py sliding_puzzle '[1-2-3][4-5-6][7-8-0]'
```
